Cheat Sheet

Every "which command/flag when" table from the course, in one scrollable page. Not a substitute for the lessons — a fast recall aid once you've actually done the labs. Grows as each Part gets written; organized in the same order as the dashboard.

Part 0 — Orientation & Terminal

CommandWhat it does
man <cmd>Full manual page for a command
<cmd> --helpQuick usage summary
whoamiCurrent logged-in user
echo $SHELLYour current shell

Part 1 — Filesystem & Files

CommandWhat it does
pwd / cd / ls -laWhere am I / move / list (including hidden files)
cp -r, mv, rm -rCopy/move/remove, recursively for directories
cat, less, head -n, tail -nDump, page, first N lines, last N lines
tail -fFollow a growing file live
chmod u+x / chmod 755Symbolic vs numeric permission change
chown user:groupChange owner and group
ln -s target linkCreate a symlink

Part 2 — Users, Processes & Services

CommandWhat it does
id, groupsCurrent UID/GID and group memberships
sudo <cmd>Run one command as root, using YOUR password
ps aux, topList / live-monitor processes
kill PID / kill -9 PIDSIGTERM (polite) / SIGKILL (forced)
jobs, fg, bg, Ctrl-ZList / foreground / background / suspend a job
nohup cmd &Survive terminal close (ignore SIGHUP)
systemctl status/start/stop/enable --nowManage a systemd service
journalctl -u <service> -fFollow a service's logs live

Part 3 — Text, Pipes & Regex

CommandWhat it does
cmd > file / >> fileRedirect stdout, overwrite / append
cmd 2>&1Redirect stderr to wherever stdout points (order matters)
a | bPipe a's stdout into b's stdin
grep -i / -v / -n / -c / -rCase-insensitive / invert / line numbers / count / recursive
grep -A/-B/-C NN lines of context after/before/both
. * + ? {n,m} ^ $Any char, 0+, 1+, 0-1, exact range, start, end
[abc] [^abc] [a-z]One of, none of, range character class
( ) | (grep -E)Grouping and alternation (unescaped in ERE)
sed 's/old/new/g'Global find-and-replace (add -i to edit in place)
awk -F: '{print $1}'Print field 1, split on custom delimiter
awk 'NR>1{print $2}'Skip header row, print field 2

Part 4 — Shell & Bash Scripting

CommandWhat it does
which cmd / type cmdShow which file PATH resolves a command to
#!/usr/bin/env bashPortable shebang (PATH-based lookup)
$1 $2 $@ $# $0Args, all args, arg count, script name
"$var"Always quote expansions to avoid word-splitting
$?Exit code of the last command (0 = success)
[[ "$a" -lt "$b" ]]Numeric comparison (use -eq/-lt/-gt, not =/<)
while read -r line; do ...; done < fileCorrect line-by-line file reading
result=$(myfunc arg)Capture a function's echoed "return value"
set -euo pipefailStop on error / unset var / failed pipe stage
trap 'cleanup' EXITGuaranteed cleanup, however the script ends
bash -x script.shTrace every command with expanded variables

CommandWhat it does

~/.ssh/config (Host/HostName/User/Port/IdentityFile)Per-server connection shortcuts
