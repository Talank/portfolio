Emacs Course

Everything you need to go from "never opened Emacs" to genuinely comfortable daily-driving it — on a Mac, specifically, which is why Part 1 is entirely about getting Meta/Super set up correctly before anything else, rather than fighting the keyboard the whole way through. Buffers, windows, the kill ring, search, your own init.el, packages via use-package, major/minor modes, Org mode, project navigation, Magit, and TRAMP (editing a remote SSH file as if it were local — the exact bridge back to the Linux course). Taught the same way as the other courses: real mechanics, a One Piece story that acts the idea out (plus Big Bang Theory and Friends scenes), interactive animated diagrams, a practice lab, a quiz, and the interview questions companies actually ask.

Every lesson teaches the essentials by default — flip Essentials → Full Depth in the header any time you want a topic's deeper mechanism, edge cases, and "why it's actually built that way." Nothing is gated behind it; it's purely how far a lesson goes.

All music and sound effects are generated live in your browser with the Web Audio API — original compositions, nothing to license. Character portraits are original flat-vector art, not traced from any show.

How to use this

  1. Do Part 1 before anything else, even if you're impatient to start editing. Every later lesson's keybindings assume Option is set up as Meta — skip it and half the muscle memory you build will be fighting macOS's own Option-key accent shortcuts instead of Emacs.
  2. In each lesson: read the concept, open the story + animation the first time through, click through the interactive walkthrough, expand any Technicality corner question you can't already answer out loud, then do the Practice Lab — type the keybinding/elisp yourself, check it in the browser, then try it for real in your own Emacs.
  3. Before marking a lesson complete, try the interview questions at the bottom out loud.
  4. Keep the Cheat Sheet open in a second tab — every keybinding table in one place.
  5. Finish with the timed Interview Drill — randomized questions from everything you studied, on a countdown.
